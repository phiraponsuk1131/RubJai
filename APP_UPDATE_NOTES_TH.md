RubJai 3.0.2 ปรับหน้าจอและสลิปให้นิ่งขึ้น

- แก้หน้าหลักไม่ให้ข้อความไทยถูกบีบจนเรียงเป็นแนวตั้งบนจอแคบ
- ปรับการ์ดสลิปรอตรวจและข้อมูลจากสลิปให้อ่านง่ายขึ้น
- ปรับการอ่านชื่อจากสลิปให้ไม่รับข้อความ OCR เพี้ยนเป็นชื่อผู้รับ
- ซิงค์สลิปย้อนหลังได้สูงสุด 1 เดือน และยังเฝ้าดูสลิปใหม่แบบเรียลไทม์ขณะเปิดแอป
- เพิ่มชุดตรวจ layout บนคอมก่อน build APK เพื่อลดโอกาส UI หลุด
- ย้ายการทดสอบ UI จริงไปทำบนเครื่องนี้หรืออุปกรณ์จริง ส่วน GitHub ใช้ build APK และตรวจเงื่อนไขหลัก

หมายเหตุ: popup นี้เป็นข้อมูลอัปเดตแอปเท่านั้น การซิงค์สลิปจะไม่เด้ง popup แจ้งผลสแกน
