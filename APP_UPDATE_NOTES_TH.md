RubJai 3.0.2 แก้ UI สลิปรอตรวจและชื่อสลิป

แก้แถบสลิปรอตรวจในหน้าหลักไม่ให้ข้อความไทยถูกบีบจนเรียงเป็นแนวตั้งบนมือถือจอแคบ

เพิ่มชุดทดสอบ layout บนคอมแบบไม่ต้องใช้สิทธิ์ admin โดยตรวจหน้าจอ 360dp และสร้างไฟล์ preview SVG ไว้ดูประกอบก่อน build

ปรับการอ่านชื่อจากสลิปให้ไม่รับข้อความ OCR เพี้ยน เช่นตัวอักษร Latin แปลก ๆ เป็นชื่อผู้รับ

ซิงค์สลิปย้อนหลังได้สูงสุด 1 เดือน และยังเฝ้าดูรูปใหม่แบบเรียลไทม์ระหว่างเปิดแอป

บังคับให้ระบบซิงค์สลิปใช้ parser key ใหม่ เพื่อให้สแกนรูปของวันนี้ซ้ำด้วยกติกาที่ปลอดภัยขึ้น

GitHub Actions จะตรวจ text, สลิปตัวอย่าง, flow UI และ layout preview ก่อน build APK

เพิ่ม smoke test บน Android emulator ใน GitHub Actions เพื่อติดตั้ง APK จริง เข้าโหมดทดลอง เปิดหน้าหลัก หน้าเพิ่มรายการ และหน้าเลือกหมวด พร้อมเก็บ screenshot ไว้ตรวจ

ขยาย smoke test ให้เปิดหน้าบัญชี โปรไฟล์ และวางแผนหนี้ด้วย พร้อมเปลี่ยนโลโก้เป็นแบบมินิมอลกว่าเดิม

เพิ่มตัวตรวจความเป็นส่วนตัวเพื่อกันไม่ให้รูปสลิปจริง รูปหน้าจอ วิดีโอ path ใน Downloads หรือเลขอ้างอิงสลิปจริงถูก commit ขึ้น GitHub
